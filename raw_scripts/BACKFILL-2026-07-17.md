## Sanitization And Limitations

- The original Velgraph principal helper embedded a bearer-token digest. The
  preserved version omits it and requires `VELGRAPH_BEARER_TOKEN_SHA256`.
- Generated plaintext tokens and generated secret-bearing configuration were
  not copied. Scripts that generate or install credentials carry warnings.
- The VEL-1123 reverse-auth verifier depended on `/tmp/velcore.proto`, which no
  longer existed during recovery. The script remains useful as historical
  evidence but is not runnable without a compatible proto.
- Several scripts contain historical hostnames, addresses, ports, and paths.
  These are provenance, not current configuration authority.

## Intentionally Excluded

- Bonsai classifier probes were excluded because Bonsai was explicitly retired
  from consideration.
- Files inside cloned repositories, package caches, build trees, test fixtures,
  generated dependency directories, and upstream Git hooks were excluded.
- Trivial command fragments and files without durable diagnostic or procedural
  value were excluded.
- Any generated output that could contain credentials was excluded.
